
### 🏁 Maratona Java – DevDojo

Repositório criado para armazenar e demonstrar as habilidades e conhecimentos adquiridos durante o curso **Maratona Java** do **DevDojo**.

📚 **Conteúdo abordado:**

* Fundamentos da linguagem Java
* Programação orientada a objetos (POO)
* Classes, métodos, herança, polimorfismo e encapsulamento
* Coleções (List, Set, Map)
* Manipulação de arquivos
* Tratamento de exceções
* Boas práticas de programação

💡 **Objetivo:**
Praticar e consolidar conceitos essenciais de Java, desenvolvendo uma base sólida para projetos futuros.
